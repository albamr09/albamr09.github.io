# Alba's Notes

Static Notes Web Page

To build the source:

```bash
hugo build
```

To run the server for development:

```bash
hugo server --logLevel debug --disableFastRender -p 1313
```

## Deploy

We use github actions to build the page and deploy the result to github pages.
