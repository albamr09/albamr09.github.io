Redux is a complex state management tool, with a single store (javascrip object) as a CDS (Central Data Storage). Components

  • Store: object that holds the state
  • Reducers: events handler that manages the state and returns the new updated state. The reducers get the arguments and return the state modified.
  • Actions: describe the event handler by the reducer and has two properties:
    • Type: is the identifier of the action
    • Payload: holds the data
  • Dispatch: is used to send actions to update the data
