Dependency Injection

We will introduce dependency injection with annotation using autowiring:

  • Spring looks for a class that matches the attribute type (call or interface) (i.e. FortuneService)
  • Spring will inject it automatically
  • If there are multiple implementations: tell Spring which specific bean to use with the Qualifier annotation
  • Constructor Injection
  • Setter Injection
  • Method Injection
  • Field Injection
  • Inject using Properties File
  • Qualifier Annotation

Which dependency to use

Choose a style and stay consistent in your project. You get the same functionality regardless of the type of dependency injection you use.
