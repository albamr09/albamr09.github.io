Explore Data Credibility

Good data is ROCCC:

  • Reliable
  • Original: Check the quality of the data with the original source
  • Comprehensive: contain all critical information needed to find a solution for a problem
  • Current
  • Cited: should be cited by trusted sources

Bad data is data that does not satisfies one or more of the characteristics listed above.

Some good sources for data are:

  • Vetted public datasets
  • Governmental agency data
  • Academic papers
